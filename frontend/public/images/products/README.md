## Supported Formats
- JPG/JPEG
- PNG
- WebP

## Recommended Image Size
- Width: 800px - 1200px
- Height: 800px - 1200px (square or portrait)
- File size: Under 500KB for faster loading

## After Adding Images
Run this command from the backend folder:
```bash
cd backend
node scripts/updateLocalImages.js
```

This will automatically update all products to use your local images!
